Definiciones

Internet
Es la red informática muncial, descentralizada, formada por la conexión directa entre ordenadores mediante un protocolo especial de comunicación.
WWW
Es la abreviación de World Wide Web el cual es un sistema interconectado de páginas web públicas accesibles a través de Internet.
HTML
Es la abreviación de HyperText Markup Language y es el lenguaje de marcado utilizado para crear y estructurar el contenido en la web. Es la base para crear páginas web y definir su formato y elementos.
