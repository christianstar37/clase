Columna 1

Este es el contenido de la primera columna. Aquí puedes añadir cualquier texto que desees. Este diseño con tres columnas es útil cuando deseas dividir el contenido en partes más pequeñas y hacer que cada sección sea fácil de leer.

Puedes organizar tu información en partes claramente separadas pero relacionadas, para que el lector pueda entender el contenido de manera más fluida.

Columna 2

Este es el contenido de la segunda columna. Las tres columnas se distribuyen uniformemente usando flexbox. Puedes ajustar el tamaño de las columnas fácilmente modificando el ancho, para que se adapten a la cantidad de contenido que quieres incluir en cada una.

Esta es otra forma de mejorar la presentación visual de la información, permitiendo que el contenido no esté amontonado en un solo bloque.

Columna 3

Este es el contenido de la tercera columna. Puedes usar esta columna para añadir más información o detalles adicionales relacionados con los temas de las primeras dos columnas. Al dividir el contenido en varias secciones, es más fácil para el usuario centrarse en una parte a la vez.

El diseño con tres columnas también es ideal para mostrar comparaciones o diferentes aspectos de un mismo tema.
